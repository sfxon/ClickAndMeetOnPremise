Freitext-Felder:

Freie Freitext-Felder

Freitextfelder können so eingestellt werden, dass sie vom Verkäufer selbst als Name-Wert Paar vergeben werden können. Damit kann der Verkäufer gewissermaßen bequem eine neue, vorher nicht vorgesehene Information hinterlegen:

{Kunden_Freitext}

Vordefinierte Freitext-Felder:

Freitextfelder können auch so definiert werden, dass der Verkäufer den Namen schon vorgegeben bekommt. Er fragt also nur noch einen Wert beim Kunden ab.


Bspw.:
{Kunden_Freitext|Fußballmannschaft} = Fragt die Fußballmanschaft ab.
{Kunden_Freitext|Hobbies} = Fragt die Hobbies des Kunden ab.

{/literal}
